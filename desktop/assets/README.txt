Place the application icon files here:

- app-icon.png  (used in development and as a generic icon)
- app-icon.ico  (Windows installer / window icon)
- app-icon.icns (macOS build, optional)

Recommended sizes:
- PNG: 1024x1024 or 512x512
- ICO: multi-size (16, 32, 48, 64, 128, 256)

Electron dev will load app-icon.png if present. Electron-builder will use app-icon.ico on Windows.
